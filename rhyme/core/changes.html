Release 1.2.0 – 2025-04-30

Type Changes By
Allow to provide custom mock/stub implementations for remote HAL API interfaces (via #withRemoteResourceOverride in RhymeBuilder and HalApiClientBuilder) ssauder
Improved consistency for handling of null values for link template variables. When calling #createLink on a client proxy, you'll now always get the link template (which may be partially expanded if some values were not null in the invocation). When following such links, any non-resolved variables will be removed before loading the target resource (rather than throwing an exception). This adds support for the following link templates where all variables are optional. ssauder
Ensure that the max-age value of failed upstream requests is also captured and taken into account for calculating the max-age of the resource being rendered. ssauder
Improved reliability when parsing max-age response header. ssauder
Simplified generics in signature of HalApiReturnTypeSupport#convertToObservable. The change is binary compatible at runtime, but (only) if you are implementing your own HalApiReturnTypeSupport, you may see a build error after upgrading the compile time dependency. ssauder
Restore old behaviour on "only null parameters" handling. Fixes 46. ckumpe

Release 1.1.0 – 2022-01-12

Type Changes By
Added functions in builder to specify custom ObjectMapper (to avoid jackson-datatype-guava dependency). ssauder
Allow methods annotated with Related to return Link instances directly. ssauder
Introduce RhymeMetadataConfiguration to opt-in for generation of embedded "rhyme:metadata" resource. ssauder
Updated dependencies for Java 17 compatibility. ssauder
Added HalApiClientBuilder and HalResponseRendererBuilder. ssauder
Introduced support for @ResourceProperty annotation. ssauder
Add caching functionality via HalResourceLoaderBuilder. ssauder
Introduced a simpler SPI interface for custom HTTP client implementations. ssauder

Type Changes By
Initial release. ssauder
